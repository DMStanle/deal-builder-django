# Deal Builder Django

This project is a simple Django web application that allows users to calculate the total cost of a product after applying a discount and tax. Users can select a product, choose a promotional deal, enter a quantity, and the application calculates the final price automatically.

The goal of the project was to demonstrate basic Django concepts including views, templates, routing, form handling, and server-side calculations.

## Instructions for Build and Use

https://youtu.be/gdgLiHt9eA4


Steps to build and/or run the software:

1. Install Python (version 3.10 +)
2. Install Django
3. Navigate to the project directory (cd DealBuilderDj)
4. Run the Django development server (python manage.py runserver)

Instructions for using the software:

1. Select a product from the dropdown menu.
2. Select a deal from the dropdown menu.
3. Enter a quantity
4. Select "Build Deal"
